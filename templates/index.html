Bienvenido al Sistema de Predicción de Símbolos

Este sistema te permite crear datasets de símbolos griegos, entrenar múltiples modelos de inteligencia artificial (Random Forest, SVM, Redes Neuronales) y predecir símbolos dibujados a mano.

Crear Dataset

Dibuja símbolos griegos (α, β, ε) para crear un conjunto de datos de entrenamiento para el modelo.

Ir a Crear Dataset

Ver Dataset

Administra las imágenes del dataset. Ve todas las imágenes creadas y elimina las que no sean necesarias.

Ir a Ver Dataset

Entrenar Modelo

Entrena modelos de aprendizaje automático (Random Forest, SVM, o Redes Neuronales) usando las imágenes recopiladas en el dataset. Compara el rendimiento de diferentes algoritmos.

Ir a Entrenar Modelo

Predecir Símbolo

Dibuja un símbolo y el modelo actualmente seleccionado intentará reconocerlo como α, β o ε. Cambia entre diferentes algoritmos para comparar resultados.

Ir a Predecir Símbolo

Instrucciones de Uso

Para obtener los mejores resultados, sigue estos pasos:

  1. Crea un dataset dibujando múltiples ejemplos de cada símbolo (α, β, ε).
  2. Selecciona y entrena diferentes modelos de IA (Random Forest, SVM, Redes Neuronales).
  3. Compara el rendimiento de los modelos y elige el que mejor funcione.
  4. Dibuja un símbolo en la página de predicción para probar el modelo seleccionado.
