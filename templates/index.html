Bienvenido al Sistema de Predicción de Símbolos

Este sistema te permite crear datasets de símbolos griegos, entrenar un modelo de inteligencia artificial y predecir símbolos dibujados a mano.

Crear Dataset

Dibuja símbolos griegos (α, β, ε) para crear un conjunto de datos de entrenamiento para el modelo.

Ir a Crear Dataset

Entrenar Modelo

Entrena un modelo de aprendizaje automático usando las imágenes recopiladas en el dataset.

Ir a Entrenar Modelo

Predecir Símbolo

Dibuja un símbolo y el modelo entrenado intentará reconocerlo como α, β o ε.

Ir a Predecir Símbolo

Instrucciones de Uso

Para obtener los mejores resultados, sigue estos pasos:

  1. Crea un dataset dibujando múltiples ejemplos de cada símbolo (α, β, ε).
  2. Entrena el modelo usando el dataset creado.
  3. Dibuja un símbolo en la página de predicción para probar el modelo.
